Installing PostGIS on Ubuntu 13.04 Raring Ringtail

By admin
on 02.08.2013

It's as simple as

sudo add-apt-repository ppa:ubuntugis/ubuntugis-unstable
sudo apt-get install postgresql-9.1-postgis-2.0-scripts

You might see alternate instructions asking you to install Sharpie's PPA, but it works only for Ubuntu Precise. The instructions in the PostGIS trac wiki is also broken ‚Äê you will get an error like so :
